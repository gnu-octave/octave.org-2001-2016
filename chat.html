The Freenode IRC service has granted the Octave project the use of the #octave channel for the purpose of chatting. Octave discussion for both developers and users takes place here. The atmosphere is relaxed and chat isn't restricted to matters strictly pertaining to Octave.

To use IRC you require an IRC client. Many options exist, and popular choices include Xchat, irssi, Chatzilla and Colloquy. Most general-purpose chat clients like Pidgin and Kopete can also be configured for IRC. Usually all you need to do is choose a nickname or change the default, connect to irc.freenode.net and join the #octave channel.

Freenode provides a web client if you prefer to not use a program external to your web browser. Just pick a nickname, fill out the captcha, and you'll be directed to the #octave channel. For more information about Freenode itself, including how to register a permanent identity within the Freenode servers using services like Nickserv, consult the documentation for Freenode.
