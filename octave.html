Introduction to Octave

GNU Octave is a high-level language, primarily intended for numerical computations. It provides a convenient command line interface for solving linear and nonlinear problems numerically, and for performing other numerical experiments. It may also be used as a batch-oriented language. For more information, see the Octave home page.

Downloading Octave

Octave can be found on in the subdirectory /gnu/octave/ on your favorite GNU mirror. For other ways to obtain Octave, please read How to get GNU Software
