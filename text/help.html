Help!

The help mailing list exists for the discussion of matters related to using and installing Octave. If would like to join the discussion, please send a message containing just subscribe in the body to help-request@octave.org. An archive of older messages is available.

Please do not send requests to be added or removed from the the mailing list, or other administrative trivia to the list itself.

If you think you have found a bug in the installation procedure, however, you should send a complete bug report for the problem to the bug-octave mailing list. This page contains more information that will help you to submit a useful report.
