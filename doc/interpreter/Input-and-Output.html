15 Input and Output

Octave supports several ways of reading and writing data to or from the prompt or a file. The most simple functions for data Input and Output (I/O) are easy to use, but only provides a limited control of how data is processed. For more control, a set of functions modelled after the C standard library are also provided by Octave.
