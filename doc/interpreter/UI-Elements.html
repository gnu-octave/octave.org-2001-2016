35.3 UI Elements

The ui* series of functions work best with the qt graphics toolkit, although some functionality is available with the fltk toolkit. There is no support for the gnuplot toolkit.

Function File: hui = uicontextmenu ("Name", value, …)
Function File: hui = uicontrol ("Name", value, …)
Function File: hui = uicontrol (parent, "Name", value, …)
Function File: uicontrol (h)
Function File: hui = uipanel ("Name", value, …)
Function File: hui = uipanel (parent, "Name", value, …)
Function File: hui = uipushtool ("Name", value, …)
Function File: hui = uipushtool (parent, "Name", value, …)
Function File: hui = uitoggletool ("Name", value, …)
Function File: hui = uitoggletool (parent, "Name", value, …)
Function File: hui = uitoolbar ("Name", value, …)
Function File: hui = uitoolbar (parent, "Name", value, …)
