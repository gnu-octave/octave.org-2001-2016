The help mailing list exists for the discussion of matters related to using and installing Octave. If would like to join the discussion, please visit the list subscription page on the web.

Messages sent to the list are also archived.

Please do not send requests to be added or removed from the the mailing list, or other administrative trivia to the list itself.

If you prefer to use a chat program instead of email, we also have the #octave IRC channel on Freenode. You can either use the Freenode web client to chat (just pick a screen name and answer the CAPTCHA) or point your preferred IRC client to irc.freenode.net and join the #octave channel.

If you think you have found a bug in Octave or in the installation procedure, however, you should submit a complete bug report to the Octave bug tracker. But before you submit a bug report, please learn how to submit a useful bug report.
