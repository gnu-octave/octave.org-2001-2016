There are several mailing lists devoted to Octave.

help: General discussion about using and installing Octave.
To see the collection of prior posting to the list, visit the help archives.
You can (un)subscribe to the list by following instructions on the list information page.

bug: Bug reports (sometimes fixes are posted to this list).
To see the collection of prior posting to the list, visit the bug archives.
You can (un)subscribe to the list by following instructions on the list information page.

sources: Enhancements contributed by Octave users.
To see the collection of prior posting to the list, visit the sources archives.
You can (un)subscribe to the list by following instructions on the list information page.

graphics: Discussion about future development of GUI and graphics tools for Octave.
Please note: this is a mailing list for discussion of future developement, not a place to ask how to use the current Octave graphics features.
To see the collection of prior posting to the list, visit the graphics archives.
You can (un)subscribe to the list by following instructions on the list information page.

maintainers: Discussion about the development and maintenance of Octave.
To see the collection of prior posting to the list, visit the maintainers archives.
You can (un)subscribe to the list by following instructions on the list information page.

info: Announcements about new releases of Octave and other news.
All postings to info-octave are also sent to the help-octave mailing list, so you do not need to subscribe to both info-octave and help-octave.
To see the collection of prior posting to the list, visit the info archives.
You can (un)subscribe to the list by following instructions on the list information page.
