There are several active mailing lists devoted to Octave.

help: General discussion about using and installing Octave.
To see the collection of prior posting to the list, visit the help archives.
You can (un)subscribe to the list by following instructions on the list information page.
maintainers: Discussion about the development and maintenance of Octave.
To see the collection of prior posting to the list, visit the maintainers archives.
You can (un)subscribe to the list by following instructions on the list information page.

Obsolete Mailing Lists

There are also some older mailing lists which are no longer active.

bug
This list was used to report bugs but is now obsolete because we are using the bug tracker on Savannah. Recent archives may be found here.
graphics
This list was used to discuss the future development of GUI and graphics tools for Octave. The maintainers list is now used for discussions about this topic. Recent archives may be found here.
sources
This list was used for posting enhancements to Octave. Now, instead of posting code to this mailing list, you are encouraged to join and contribute your code to the Octave Forge project. However, code to fix bugs in Octave itself or new core functions for Octave should be posted to the bug or maintainers lists. Recent archives may be found here.
